# Быстрый старт

## Вход в систему

**Учетные данные администратора по умолчанию:**
- Логин: `admin`
- Пароль: `admin123`

## Настройка для работы на VPS

После первого входа:

1. Нажмите на иконку пользователя в правом верхнем углу
2. Выберите "Администрирование"
3. Перейдите на вкладку "Настройки VPS"
4. Нажмите кнопку **"Авто"** - система автоматически определит адрес
5. Нажмите **"Сохранить и применить настройки"**

Готово! Сайт настроен для работы на вашем сервере.

## Изменение пароля администратора

1. Откройте "Администрирование"
2. Вкладка "Учетные данные"
3. Введите текущий пароль: `admin123`
4. Укажите новый логин и/или пароль
5. Сохраните изменения

⚠️ **Важно:** Сохраните новые учетные данные в надежном месте!
